2. Reprendre ou créer un profil local

Pourquoi charger un CV s’il n’est pas lu automatiquement ?
Parce qu’un CV final sérieux doit rester rattaché à une source. Dans cette version statique, le CV sert de preuve de continuité et de garde-fou anti-invention. Pour tester vite, chargez seulement le RAW ; pour générer un pack sérieux, chargez CV + RAW.

Règle simple : RAW seul = analyse prudente. CV + RAW = livrables autorisés. CV seul = source, mais pas de mémoire exploitable.
