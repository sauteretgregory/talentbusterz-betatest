Profil à initialiser — TalentBusterZ

Template de référence pour créer un profil candidat statique harmonisé. Aucun profil métier n’est prérempli.

Règle centrale V0.8.1 : le mode découverte peut fonctionner sans CV pour les curieux, mais il ne produit qu’un RAW démo progressif. Les livrables sérieux exigent un CV source et un RAW pro. Un RAW chargé seul permet une analyse locale prudente, mais pas un CV final complet. En phase test, l’opérateur reste responsable de l’anonymisation : aucun CV source réel ni coordonnée ne doit être publié dans GitHub.

1. Choisir le parcours

Découverte sans CV

Pour un visiteur curieux : score très prudent, questions progressives, RAW démo téléchargeable.

RAW local

Pour un bêta-testeur : chargez le RAW envoyé par Grégory. Aucune donnée candidat n’est hébergée dans ce site.

Mode pro candidat

CV source + dernier RAW connu. C’est le mode de continuité locale sérieuse.

Statut : aucun mode sélectionné.

2. Entrées du mode pro

RAW JSON : base de données locale provisoire + mémoire IA statique. À conserver avec le CV source pour les prochaines analyses.

Statut de continuité

Aucune mémoire active.

Le Data Gate ne bloque pas la curiosité. Il bloque seulement les livrables sérieux si les données ne sont pas suffisantes.

Démo statique — lecture d’une fiche de poste

Collez une offre ou quelques mots-clés. Le score distingue l’alignement estimé et la confiance des données.
