# TalentBusterZ — betatest V0.8.4 hotfix step 1

## Objet de cette version

Cette version corrige uniquement le problème prioritaire de Data Gate :

**RAW seul = livrables de travail autorisés.**

Le CV source ne bloque plus les livrables de travail.
Il ne sert qu'à produire un pack final source-vérifié.

## Livrables autorisés en RAW seul

Quand un RAW est chargé et qu'une offre est analysée, le bouton **Générer les livrables de travail** doit produire :

1. CV Word de travail + RAW actualisé
2. Note/scoring TalentBusterZ + RAW actualisé
3. Points forts / axes d’amélioration + RAW actualisé
4. RTOU + RAW actualisé

## Pack final source-vérifié

Le pack final source-vérifié reste indisponible tant que le CV source n'est pas chargé.
