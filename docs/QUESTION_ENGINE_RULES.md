# Question Engine — V0.8.4 hotfix

Cette version ne corrige pas encore toute la qualité conversationnelle du Question Engine.

Règle cible à garder pour l'étape suivante :

TBZ doit mener un entretien de clarification, pas un formulaire.

Chaque question doit :
- rappeler l'offre active ;
- rappeler la preuve active ;
- expliquer ce qui a été compris ;
- valoriser ce qui est solide ;
- indiquer ce qui manque ;
- expliquer le score ;
- inviter l'utilisateur à corriger ;
- poser une suite logique.
