# Question Engine V0.8.2

## Règle centrale

TBZ ne remplit pas un formulaire : TBZ mène un entretien de clarification.

Chaque question doit :

1. rappeler l'offre active ;
2. rappeler la preuve active ;
3. expliquer ce que TBZ a compris ;
4. valoriser ce qui est solide ;
5. indiquer ce qui manque pour le maximum ;
6. expliquer le score ;
7. inviter l'utilisateur à corriger ;
8. poser une suite logique.

## Interdits

- Questions vagues du type “Quel point faible faut-il préparer ?”
- Score qui passe de 93 à 95 sans explication.
- “Sur cette preuve...” sans rappeler la preuve.
- Bloquer le Question Engine parce qu'un CV source est manquant.
