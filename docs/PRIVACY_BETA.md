# Privacy bêta

Cette version est un démonstrateur public générique. Elle ne doit héberger aucune donnée candidat.

Tant que les briques sécurité/RGPD ne sont pas en place :

- ne pas publier de CV source réel ;
- ne pas publier de coordonnées, nom complet, adresse, téléphone, email ;
- ne pas publier de RAW nominatif ou pseudonymisé détaillé ;
- ne pas utiliser GitHub comme espace de backup candidat ;
- conserver les sources réelles hors repo GitHub ;
- les bêta-testeurs chargent leur RAW localement dans leur navigateur.

Le fichier RAW est une mémoire locale portable. Il peut contenir des données professionnelles ré-identifiantes même sans coordonnées directes.
