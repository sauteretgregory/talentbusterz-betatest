# Privacy bêta

Tant que les briques sécurité/RGPD ne sont pas en place :

- ne pas publier de CV source réel ;
- ne pas publier coordonnées, nom complet, adresse, téléphone, email ;
- ne pas publier de RAW nominatif complet ;
- utiliser un profil anonymisé préparé par l’opérateur ;
- conserver les sources réelles hors repo GitHub.
