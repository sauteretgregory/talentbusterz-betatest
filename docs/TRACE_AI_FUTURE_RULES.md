# Trace IA future — V0.8.4 hotfix step 2

## Règle centrale

Le RAW est une mémoire de traçabilité, pas un CV final.

Il ne doit pas filtrer l’information utilisateur.

## À conserver

Chaque interaction doit tracer :

- offre active ;
- preuve active ;
- question posée ;
- réponse brute utilisateur ;
- commentaire utilisateur sur la question ou le score ;
- interprétation assistant ;
- score avant ;
- score après ;
- gain obtenu ;
- gain potentiel ;
- éléments appréciés ;
- éléments manquants ;
- points de prudence ;
- décision d’usage ;
- métadonnées de génération.

## Principe

La sélection se fait au moment de générer le CV.
La suppression du RAW ne doit jamais être utilisée pour “nettoyer” une réponse utile.
